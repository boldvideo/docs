Bold's AI Coaching feature helps you create better video content with intelligent feedback and suggestions.

## Features

- **Content Analysis**: AI analyzes your video content for clarity and engagement
- **Speech Feedback**: Get suggestions on pacing, tone, and delivery
- **Visual Recommendations**: Improve lighting, framing, and composition

## Getting Started

1. Upload a video to your Bold account
2. Navigate to the AI Coaching section
3. Select the type of feedback you want
4. Review and apply the suggestions

## API Access

You can also access AI Coaching features via the API. See the [AI Ask endpoint](/api/ai-ask) for details.
