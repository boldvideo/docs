This guide covers how to upload videos to the Bold platform.

## Supported Formats

Bold supports the following video formats:

- MP4
- MOV
- WebM
- AVI

## Upload Methods

### Web Upload

You can upload videos directly through the Bold dashboard.

### API Upload

For programmatic uploads, use the Bold API. See the [API Reference](/api) for details.

### SDK Upload

Use the Bold SDK for seamless integration. See the [SDK Documentation](/sdk) for more information.
